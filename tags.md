# Référence des tags

Les tags utilisent un préfixe pour indiquer leur famille :
- `c-` : Contenu
- `f-` : Format
- `o-` : Outil
- `p-` : Pédagogie

---

## Contenu (`c-`)

Tags transversaux qui couvrent plusieurs sous-chapitres ou thèmes.

| Tag | Description |
|-----|-------------|
| `c-livrets` |   Livrets |
| `c-multiplication` |   Multiplication |
| `c-division` |   Division |
| `c-criteres-divisibilite` |   Critères de divisibilité |
| `c-fractions` | Fractions |
| `c-probabilites` | Probabilités |
| `c-statistiques` | Statistiques |
| `c-equations` | Équations |
| `c-fonctions` | Fonctions |
| `c-geometrie` | Géométrie |
| `c-trigonometrie` | Trigonométrie |
| `c-nombres-premiers` | Nombres premiers |
| `c-priorite-operations` | Priorité des opérations |

---

## Format (`f-`)

Comment la ressource est utilisée en classe.

| Tag | Description |
|-----|-------------|
| `f-groupe` | Travail en groupe |
| `f-individuel` | Travail individuel |
| `f-commun` | Activité avec toute la classe |
| `f-maison` | Travail à la maison |
| `f-station` | Travail en station |
| `f-oral` | Activité orale |
| `f-concours` | Format concours / compétition |
| `f-defi` | Défi rapide en début de cours |
| `f-projet` | Travail sur plusieurs séances |
| `f-sortie` | Activité hors classe |
| `f-TICE` | Activité connectée |

|-----|-------------|

| `p-modelisation` | Modéliser une situation |
| `p-differentiation` | Activité différenciée selon les niveaux des élèves |

---

## Ajouter un nouveau tag

1. Choisis le bon préfixe (`c-`, `f-`, `o-`, `p-`)
2. Ajoute-le dans ce fichier
3. Utilise-le directement dans les métadonnées de ton `.tex` — **aucune modification du site nécessaire**

Les tags apparaissent automatiquement dans la sidebar dès qu'ils sont présents dans au moins une ressource.
